# Notes to Deck - Features

Transform notes, images, and ideas into polished PDF decks in seconds.

## Core Features

- **Content Input** - Title, notes textarea, drag-drop image upload with gallery
- **Template Selection** - 3 built-in templates + AI-generated custom templates
- **Live Preview** - Real-time A4 preview with selected template applied
- **PDF Export** - One-click download with auto-generated filename

## Built-in Templates

1. **Bold Editorial** - Dark, dramatic magazine-style layout
2. **Photo Spread** - Image-centric with gradient backgrounds
3. **Clean Deck** - Professional two-column (notes left, images right)

## Design Studio (Advanced)

- Upload brand images for automatic color extraction
- AI-powered mood detection (temperature, saturation, brightness)
- Generate custom templates using Claude AI
- Manual color adjustments and saving

## History

- Auto-saves export metadata (last 50 exports)
- Delete individual items or clear all

## Privacy

- All content processing happens in browser
- No backend storage - data stays on your device
- Optional AI features with graceful local fallback
